# woowahan-mission-submit

우아한테크코스 미션 제출을 도와주는 사이트

## 사용법

`header` 에서 `step`을 고른 후 닉네임과 레포 주소를 적으면 그에 해당하는 커맨드를 알려줍니다.

FORK와 PR을 간편히 할 수 있게 도와줍니다.

레포 주소는

> https://github.com/iftype/woowahan-mission-submit
> https://github.com/iftype/woowahan-mission-submit.git

둘 다 가능합니다
